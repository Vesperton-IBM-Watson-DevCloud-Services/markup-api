Markup API Demo

Description

This is a demo of the Markup API.

When the page loads, a call is made to the API which returns back HTML and CSS. Each are then appended to the page automatically. In this demo, the masthead above (including the navigation elements, breadcrumbs, and search) is being rendered from data received by the API.

Inspecting the page source shows that the masthead elements are not initially present.


Implementation

The API is called through a script tag using JSONP. A small JavaScript library is included to append the response data directly into the page. A div with the id of "target" is used as a container for the new HTML.
